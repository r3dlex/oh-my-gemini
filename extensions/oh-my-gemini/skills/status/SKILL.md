# Status Skill (oh-my-gemini)

Use this skill to checkpoint current progress.

## Include
- current objective
- completed work
- work in progress
- blockers or risks
- next actions in order

## Helpful commands
- `omg team status --team <name>`
- `omg sessions`
- `omg cost`
- `omg skill handoff`
