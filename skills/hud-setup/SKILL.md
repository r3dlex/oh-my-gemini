# HUD Setup Skill (oh-my-gemini)

## Quick Start

- Choose a HUD preset, verify the render, then note any preferred watch-mode workflow.

Use this skill when the user wants to view or configure the HUD.

## Quick Start
- `omg hud`
- `omg hud --watch`
- `omg hud --preset minimal`
- `omg hud --preset focused`
- `omg hud --preset full`

## Guide the user to
1. choose a preset
2. verify current context renders correctly
3. use watch mode for live monitoring
4. document any preferred preset or workflow notes
