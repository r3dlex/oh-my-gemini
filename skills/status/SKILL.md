# Status Skill (oh-my-gemini)

## Quick Start

- Summarize the objective, completed work, blockers, and ordered next actions.

Use this skill to checkpoint current progress.

## Quick Start
- current objective
- completed work
- work in progress
- blockers or risks
- next actions in order
