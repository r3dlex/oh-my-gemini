# Status Skill (oh-my-product)

Use this skill to checkpoint current progress.

## Include
- current objective
- completed work
- work in progress
- blockers or risks
- next actions in order

## Helpful commands
- `omp team status --team <name>`
- `omp sessions`
- `omp cost`
- `omp skill handoff`
