# Debug Skill (oh-my-product)

## Quick Start

- Reproduce the smallest failing case, isolate the cause, apply a minimal fix, then rerun validation.

Use this skill for regressions, failing tests, broken CLI behavior, or suspicious state.

## Quick Start
1. Reproduce the problem or gather the smallest failing evidence.
2. Narrow the cause with file, command, and output evidence.
3. Apply the minimum credible fix.
4. Re-run the relevant validation.

## Expected output
- failure summary
- root cause
- files touched
- validation evidence
- residual risk
